## Design Context

### Users
Personal use by the creator. A screenshot tool for quick, casual captures during daily work and browsing.

### Brand Personality
- **Fast**: Instant capture, minimal friction
- **Focused**: No distractions, purposeful UI
- **Approachable**: Warm, not intimidating, easy to use

### Aesthetic Direction
**"Refined Industrial Dark"**
- Dark mode interface (warm charcoal base, not cold blue-gray)
- Minimalist screenshot tool aesthetic (inspired by ShareX but more refined)
- Industrial/utilitarian feel with approachable warmth
- Professional creative tool vibe

### Visual Language
- **Colors**: Deep warm charcoal (#2d2a26) with amber accent (#e8a33c)
- **Typography**: Geist Sans + Geist Mono (technical but warm, distinctive)
- **Spacing**: 4pt base scale, generous breathing room
- **Motion**: Smooth expo easing, purposeful transitions
- **Shadows**: Subtle, warm-tinted depth

### Design Principles
1. **Calm over excitement** - Interface should fade into background
2. **Purposeful density** - Every element earns its place
3. **Warm technicality** - Precise but not cold
4. **Immediate feedback** - Actions feel responsive and alive
5. **Consistent rhythm** - Visual patterns repeat predictably

### The Unforgettable Element
The "breathing" canvas - screenshots appear to float in a purposeful, calm space with subtle ambient motion.

### Anti-References
- No neon cyberpunk aesthetics
- No glassmorphism for decoration
- No gradient text or side-stripes
- No bounce/elastic animations
